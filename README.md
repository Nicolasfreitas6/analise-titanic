# Análise Exploratória - Titanic
Projeto/análise desenvolvida para o curso de Introdução ao Data Science do SCTEC e Senai.

## Objetivo
Realizar análise exploratória de dados (EDA) no dataset Titanic, incluindo:
- Tratamento de dados;
- Criação de variáveis;
- Análises estatísticas;
- Visualizações gráficas.

## Tecnologias
- Python;
- Pandas;
- NumPy;
- Matplotlib;
- Seaborn.
